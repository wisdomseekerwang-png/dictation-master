# 听写大师 (DictationMaster) - iPad听写应用

## 项目概述
为一个用户创建了一个iPad听写应用，支持：
- 通过PDF、TXT、粘贴文本导入词库
- 错词本管理（自动+手动添加）
- 可配置的听写功能（词数、速度、重复次数、间隔）
- 语音合成朗读

## 技术栈
- React + TypeScript + Vite
- TailwindCSS
- Web Speech API
- pdf.js (PDF解析)
